columbo_design.beacon.design_beacon(beacon_site: str, stem_len: int = 11, loop_len: int = 9) → str[source]

Design a DNA molecular beacon sequence from a target site.

Beacon = [stem1 | loop | stem2], where:

  • stem1 = first stem_len nt of the displaced strand reverse

  • loop = complementary of central region

  • stem2 = inverse complement of stem1

Parameters:

  • beacon_site (str) – target region sequence (displaced strand)

  • stem_len (int) – length of each stem arm

  • loop_len (int) – length of the loop (unused, inferred from site)

Returns:

designed beacon DNA sequence

Return type:

str

columbo_design.beacon.hairpin_correct(struct: str, ideal_stem=11, ideal_loop: int = 9) → float[source]

Evaluate how close a folded beacon hairpin matches the ideal stem-loop-stem pattern.

The function checks the lengths of the initial run of ‘(’ (stem1), the central run of ‘.’ (loop), and the final run of ‘)’ (stem2). Each is normalized by its ideal length and averaged to produce a score in [0,1].

Parameters:

  • struct (str) – RNA secondary structure in dot-bracket notation

  • ideal_stem (int) – ideal number of base pairs in each stem

  • ideal_loop (int) – ideal number of unpaired bases in the loop

Returns:

average normalized score of (stem1, loop, stem2)

Return type:

float

columbo_design.beacon.score_beacon(beacon_seq: str, beacon_site: str, gRNA_seq: str, tm_floor: float = 50.0) → float[source]

Compute the overall molecular beacon performance score and its components.

Combines:

  1. hairpin structural correctness

  2. melting temperature activation F_tm

  3. complementarity to displaced strand R_bn

  4. lack of complementarity to gRNA R_gr

  5. hybridization free energy score F_e

Parameters:

  • beacon_seq (str) – beacon DNA sequence

  • beacon_site (str) – displaced target site sequence

  • gRNA_seq (str) – guide RNA sequence (protospacer + tracr)

  • tm_floor (float) – minimum acceptable Tm for activation

Returns:

tuple (global_score, R_bn, R_gr, F_tm, F_e, hairpin_score)

Return type:

(float, float, float, float, float, float)

columbo_design.cli.main() → None[source]

Main entry point for the Columbo CLI tool.

  1. Parses command-line arguments.

  2. Loads sequences from the given FASTA.

  3. Finds NGG PAM sites in each sequence.

  4. Builds ColumboParts objects and ranks top candidates.

  5. Designs molecular beacons and primers around each PAM.

  6. Writes results to JSON or pickle files.

  7. Prints a summary report to stdout.

Raises:

ValueError – If flanking window around a PAM goes out of sequence bounds.

Design forward and reverse primers around a CRISPR PAM site using Primer3.

This function takes a DNA sequence and the zero-based index of the PAM site, then calls Primer3 to select primers flanking the protospacer region (25 nt upstream + PAM).

Parameters:

  • seq (str) – Full DNA sequence to search for primers.

  • pam_pos (int) – Zero-based index of the first base of the PAM motif in seq.

Returns:

Primer3 raw output dictionary containing primer positions and metrics.

dict

  • RuntimeError – When Primer3 fails to find any primer pairs.
