class core.ColumboParts(sequence: Seq, position: int)[fuente]¶

Bases: object

Clase donde se almacenan las instancias correspondientes al score y tm

Parámetros:

  • sequence (Seq) – Objeto Seq a analizar

  • position (int) – Posición de la repetición NGG (sitio PAM)

Función Constructora donde sequence es un objeto seq y la posición un número entero donde se encuentra cada nt si la posición + 3 es mayor que la longitud de la secuencia

Muestra:

TypeError – Error obj Seq &/o int no reconocido

..note::

Este programa solo admite objetos Seq: .fasta

..warning::

No usar floats
