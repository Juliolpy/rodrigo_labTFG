modsum module¶

modsum.modsum(a: int, b: int, c: int = 1) → int[fuente]¶

Realiza la suma en módulo c de dos números a y b.

En caso de que el usuario no proporcione la base c, devuelve la suma habitual.

Parámetros:

  • a (int) – Primer número a sumar.

  • b (int) – Segundo número a sumar.

  • c – Base para calcular el módulo. Default: 1.
